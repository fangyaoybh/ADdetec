阿尔茨海默病 (简称AD) 是一种进行性神经退行性疾病，会严重损害认知功能，导致记忆力减退和其他行为改变。AD的早期诊断和干预可以延缓疾病进展，因此成为研究关注的焦点。传统的临床AD检测方法往往在患者出现明显症状时前去医院进行检查，存在成本高、耗时长、依靠医生主观经验等问题，不适合大规模的早期排查。
深度学习模型通过检测人类医生可能遗漏的细微模式和生物标志，可以识别出阿尔茨海默病的早期迹象，从而及时进行干预，这种早期发现可以改善治疗结果。然而深度学习在AD检测的应用上存在着以下挑战： 
一：早期检测能力有限。现有深度学习模型以医学影像作为检测的依据，而医学影像往往在患者出现明显症状时才会得到应用，不利于疾病初期的检查。
二：缺乏动态监测能力。AD的检测和治疗是一个动态分析过程，而现有模型为“单次快照式”检测，无法对病情实现长期跟踪。
最新的研究将语言功能障碍作为认知能力下降的检测依据，因为语音模式的细微变化，例如流畅度降低、发音模糊等，可以作为检测的早期指标，为早期诊断提供了一种非侵入性且具有成本效益的方法。
基于此，我们使用语音数据代替医学影像进行检测。并基于Mindspore框架设计了一个文本和声学特征的双模态检测模型来获取更加丰富的语言特征，可以在病情前期就检测出患者的异常表现。
针对缺乏动态监测能力的问题，我们开发了一个家用对话机器人，通过机器人持续获得老人的语言数据，并交由模型进行分类，从而实现长期的病情检测与预警。
系统主要有三个模块组成。在开发板设备端实现一个语音对话机器人，用于与老人日常交互，并根据对话数据调用分类模型进行结果预测。华为云存储模型预测结果，进行数据统计，用于数据流转。手机移动端可以随时获取云端存储的数据，便于使用者监测老人的状态。
